# Sign In Assessment with AngularJS for Epicodus

### Connor Abdelnoor

###### April 24, 2015

#### Description

This app will allow a user to sign in existing students, and view lists of students who are and aren't signed in.

#### Set Up

1. `git clone` the repository
2. `cd` into the project directory
3. Run a local server of your choice in the project directory
4. Navigate to `localhost:8000` in your preferred browser

#### Technologies

* AngularJS
* Bootstrap
